# v0.1 Release Notes Draft

PermissionPilot v0.1 will be the first public preview of the macOS permissions dashboard.

## Highlights

- Native SwiftUI macOS app.
- Code signing identity context for installed apps.
- Permission overview with sensitivity labels and grouped granted, denied, and unknown evidence.
- Best-effort TCC record matching for known macOS permission services, including typed evidence for unreadable databases, missing records, query failures, and legacy authorization columns.
- Conservative review-priority signals without malware verdicts.
- Educational explanations.
- System Settings shortcuts.
- First-run and empty-state guidance for limited macOS visibility.
- LaunchAgent, LaunchDaemon, login item, background task, ServiceManagement record, and privileged helper tool scanning.
- Background item details with stale reasons and source evidence.
- Full and filtered local Markdown and JSON report export.

## Known Limitations

- macOS may prevent direct inspection of some permission states.
- Permission states may be shown as unknown when macOS blocks access to TCC data or no matching record exists.
- Login item and background task data depends on current `sfltool dumpbtm` output.
- Privileged helper stale detection is a heuristic review signal.
- Helper-tool detection may require conservative wording to avoid false positives.
