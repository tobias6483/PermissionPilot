# Support

PermissionPilot is early-stage open source software.

## Getting Help

Use GitHub issues for:

- Reproducible bugs.
- Feature requests.
- Documentation problems.
- macOS version compatibility reports.

Please do not include secrets, private app inventories, full exported permission reports, or device identifiers in public issues.

## Security Reports

See [SECURITY.md](SECURITY.md) for vulnerability reporting guidance.
