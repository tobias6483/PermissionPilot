# Code Of Conduct

## Expected Behavior

- Be respectful and constructive.
- Assume good intent while staying precise about technical risk.
- Keep privacy and security discussions careful and evidence-based.
- Help new contributors understand macOS constraints without gatekeeping.

## Unacceptable Behavior

- Harassment, threats, or discriminatory language.
- Publishing another person's private data or local permission reports.
- Encouraging users to bypass macOS security protections.
- Posting exploit details publicly before maintainers have a chance to respond.

## Enforcement

Maintainers may remove comments, close issues, restrict participation, or take other reasonable action to protect the project and its community.
